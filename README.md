C# Projects

These are some small projects I made using C# and the .NET Framework.

# Projects:
 - Car Insurance
 - Student Database (Code First)

---

- Car Insurance

This project is an MVC web application that mimics a car insurance website. It takes user input on policy information and calculates a quote based on business logic. There is also an admin page that allows staff to view all of the quotes that have been issued.

---

- Student Database (Code First)

This project is a C# console application that demonstrates the Entity Framework Code-First approach. The application defines a Student model and uses Entity Framework to automatically generate the database and table structure. When the program runs, it creates the database and inserts a student record.
